# Heartnotes website

Homepage of the [Heartnotes](https://github.com/hiddentao/heartnotes) app.

##Â Development

Requires **Node 0.12+**.

Install:

```bash
$ npm install -g gulp
$ npm install
```

Run server with auto-reload (without minification):

```bash
$ gulp
```

To force minification of assets (build takes a bit longer) include the `--minifed` flag:

```bash
$ gulp --minified
```
